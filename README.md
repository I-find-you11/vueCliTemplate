# vueCliTemplate

### 介绍
这是一个基于 `vue cli4.x` 的二次脚手架封装模板

### 目前已经拥有的功能
- [x] 路由懒加载
- [x] 根据环境的不同配置
- [x] 接口代理
- [x] 接口配置
- [x] vuex配置
- [x] less配置
- [x] 组件全局化注册实例
- [x] 一些简单的工具函数（慢慢添加）

如果有其他需要可提交 lssues

## 安装依赖
```
npm install
```

### 运行在开发环境
```
npm run serve
```

### 运行在开发环境的测试库
```
npm run serve-test
```

### 运行在生产环境
```
npm run build
```

### 运行在生产环境的测试库
```
npm run serve-test
```
